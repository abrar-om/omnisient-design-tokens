# Omniscient Design Tokens

Source of truth for design tokens used by `omniscient-ui-original` and `omniscient-ui-enhanced`.

## Structure

- `tokens/source/` — Figma exports (W3C Design Tokens format or Tokens Studio JSON)
- `build/` — Compiled outputs (gitignored, generated by build script)
- `build.mjs` — Style Dictionary build configuration

## Build

```
npm run build
```

Outputs:
- `build/css/tokens.css` — CSS custom properties
- `build/tailwind/tokens.json` — flat JSON for Tailwind config
- `build/js/tokens.js` — ES6 module for direct imports
